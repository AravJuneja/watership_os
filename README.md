# What is watership OS 

Watership OS is essentially just my dotfiles and config setup for my laptop that I have been working on for the past couple years. I decided to package it as an ISO because I was bored and thought it would be cool to have my whole setup on a USB.

Watership OS is based on Ubuntu 24.04.

The goal of Watership is a setup that does not make you feel like you are using an OS created by someone else.

It's like the computers in movies. No brand, no clutter, no sense that someone else built the thing you're sitting at. Just a screen and what you're doing on it.

# Default Keybinds ?!
Now that you have watership OS installed you will find that you are greeted to a blank white screen. Everything on Watership OS can (and should) be done using just the keyboard except for some edge case scenarios.
